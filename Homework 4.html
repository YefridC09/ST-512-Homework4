Problem 2

batch <- factor(rep(1:5, each = 3))
treatment <- factor(rep(c("A", "B", "C"), times = 5)) 
strength <- c(52, 60, 56, 47, 55, 48, 44, 49, 45, 51, 52, 44, 42, 43, 38) 
data <- data.frame(batch, treatment, strength)

a

For this randomized complete block design it is recognize that there are 3 drying processes (\(a = 3\) treatments), and 5 batches (\(b = 5\) blocks). A suitable model for this experiment is shown below:

\(y_{ij} = \mu + \alpha_i + \beta_j + \epsilon_{ij}\)

Where:

\(i = 1, 2, 3\) number of treatments
\(j = 1, 2, 3, 4, 5\) number of blocks
\(y_{ij} =\) the observation in the \(j^{th}\) block receiving the \(i^{th}\) treatment
\(\mu =\) overall mean
\(\alpha_i =\) the effect of the \(i^{th}\) treatment. \(\sum_{i=1}^n \alpha_i = 0\)
\(\beta_j =\) the effect of the \(j^{th}\) block. \(\sum_{i=1}^n \beta_i = 0\)
\(\epsilon_{ij} =\) random error \(\sim N(0, \sigma^2)\)

b

To evaluate the if there is a statistical difference between the treatments, it is necessary to produce an ANOVA table including the blocks and analyze the p-value of the treatment

model <- aov(strength~treatment+batch, data = data)

Based on the p-value(0.014) that is smaller that our assumed \(\alpha = 0.05\), we have strong evidence to conclude that at least one of the treatment’s mean strength is different than the others.

The residuals vs fitted plot, shows that based on the random position of all the data across the the graph, we can conclude that the data meet the equal variance assumption(not funnel shape).
